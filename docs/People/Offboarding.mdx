# Offboarding

<ControlledDocBanner />

The manager of the person leaving OET must ensure that by the end of their last day:
- The person's OET Google account (email) is deactivated (ask TODO if you need support with this)
- The person's account is removed from the OET GitHub [people page](https://github.com/orgs/open-energy-transition/people) (ask Siddharth Krishna if you need support with this)
- The person's solver license keys are deactivated (ask Siddharth Krishna if you need support with this)
