# Product Principles
Open Energy Transition continues to build on open software, initially started in Academia, back in 2016. As of now, the project counts more than 100,000 lines of Python code.

The code gets reviewed & modified by dozens of associates/contributors, which of most are PhDs, or PhD Students, in power systems planning, operation & energy economics.

## Code Extensions

PyPSA modules are further improved, as needed, for the pruposes of the projects being conducted in the different geographical areas, and the aforementioned consultation services.

## Code Review & Approval

All of the available code regarding Open Energy Transition and PyPSA is free for use, as well as further modifications. The modifications can be submitted upstream to GitHub, using GitHub Fork -> Pull Request. Upon the review, the code maintainer acts on the basis of proposed code change:
- accepts the change,
- comments on the change,
- rejects the change, providing a reason.
